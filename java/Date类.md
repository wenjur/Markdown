# Date类

java.util.Date表示日期和时间的类，Date类精确到毫秒值，表示一个特定的事件点，比如：2018-08-08 09:56:33:234

##### 日期转换为毫秒

* 当前的日期：2020-09-29

* 时间原点：1970年1月1日 00:00:00

即计算时间原点到当前日期经过的毫秒数，是一个long类型的值。

使用方法`System.currenTimeMillis()`即可获取

## 构造方法

`Date()`

`Date(long date)`

## 成员方法

`long getTime()`

返回自时间原点以来date对象标识的毫秒值
